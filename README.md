# GameMode.Common
A collection of reusable utilities and abstractions for Open Multiplayer gamemodes built with [SampSharp](https://github.com/ikkentim/SampSharp).

GameMode.Common was originally extracted from [Capture The Flag](https://github.com/DevD4v3/Capture-The-Flag) to host functionality that is not specific to a single gamemode. It contains reusable code intended to be shared across multiple gamemodes and projects built with [SampSharp](https://github.com/ikkentim/SampSharp).

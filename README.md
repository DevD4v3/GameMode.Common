# GameMode.Common
A collection of reusable utilities and abstractions for [open.mp](https://github.com/openmultiplayer/open.mp) (Open Multiplayer, a multiplayer mod for GTA San Andreas) game modes built with C# and the [SampSharp](https://github.com/ikkentim/SampSharp) framework.

GameMode.Common was originally extracted from [Capture the Flag](https://github.com/DevD4v3/Capture-The-Flag) to host functionality that is not specific to a single game mode. It contains reusable code intended to be shared across multiple game modes and projects built with SampSharp.
